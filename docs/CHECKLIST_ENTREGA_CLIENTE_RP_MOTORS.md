# Checklist de entrega al cliente · RP Motors

> Objetivo: entregar el sitio web listo para operar, con claridad comercial, técnica y operativa.

## 1) Identidad y marca
- [ ] Nombre comercial correcto: **RP Motors**.
- [ ] Textos principales aprobados por el cliente.
- [ ] Datos de contacto validados (WhatsApp, teléfono, dirección).
- [ ] Horarios de atención confirmados.

## 2) Contenido e inventario
- [ ] Vehículos cargados con nombre/versiones correctas.
- [ ] Precio, año y kilometraje correctos por unidad.
- [ ] Fotos reales asignadas al vehículo correcto.
- [ ] Vehículos destacados configurados.
- [ ] Reseñas visibles en formato discreto (sin saturar la Home).

## 3) Conversión (ventas)
- [ ] Botón fijo de WhatsApp funcionando.
- [ ] Botones “Consultar” / “Solicitar ficha” funcionando.
- [ ] Click to call funcionando en móvil.
- [ ] Mapa y dirección visibles.

## 4) Panel admin
- [ ] Ruta admin accesible: `/admin`.
- [ ] Alta de vehículo funcionando.
- [ ] Actualización de foto por vehículo funcionando.
- [ ] Carga de datos por formulario funcionando.
- [ ] Clave de admin definida y entregada al dueño.

## 5) Seguridad mínima
- [ ] `ADMIN_SECRET` cambiado a clave fuerte.
- [ ] Claves entregadas por canal privado.
- [ ] Acceso al hosting/documentación centralizado.

## 6) Dominio y publicación
- [ ] Dominio conectado (si aplica).
- [ ] Certificado HTTPS activo.
- [ ] Sitio publicado en producción.
- [ ] Redirección de www/no-www definida.

## 7) Operación diaria
- [ ] Responsable interno definido (quién carga autos).
- [ ] Frecuencia de actualización acordada (diaria/semanal).
- [ ] Política de vehículos vendidos (ocultar/quitar).
- [ ] Protocolo de respuesta por WhatsApp acordado.

## 8) Entregables al cliente
- [ ] Manual de uso entregado.
- [ ] Video corto de operación (opcional, recomendado).
- [ ] Contacto de soporte post-entrega definido.
- [ ] Alcance de garantía/soporte por escrito.

## 9) Aceptación final
- [ ] Aprobación visual del dueño.
- [ ] Aprobación funcional del dueño.
- [ ] Fecha de pase a producción confirmada.
- [ ] Acta de entrega cerrada.

---

## Recomendación comercial
Para la reunión final, mostrar en vivo:
1. Home pública.
2. Carga de un vehículo nuevo en `/admin`.
3. Refresco y visualización inmediata en Home.

Esto acelera cierre porque demuestra autonomía real del negocio.
